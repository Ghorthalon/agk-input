Class: KeyboardInput

KeyboardInput()

Simple class to help with Keyboard Input. This class adds helper functions to pole the state of a key without firing for events. This way, you can check for key presses directly inside your game loop. It adds functions for isJustPressed() and isJustReleased(), which only return true once and false un subsequent calls until the key has either been pressed or released again.
